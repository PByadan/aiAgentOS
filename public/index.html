aiAgentOS

AI Agent Operating System v1.0

模块化 AI Agent 操作系统框架。基于 Node.js 构建，提供 Agent 注册中心、事件总线、 沙箱隔离执行、统一中间件管道，支持快速搭建 AI 驱动的业务系统。

打开控制台 查看架构

核心特性

⚙

Agent 注册中心

统一的 Agent 注册中心，支持注册、发现、执行、启停管理。每个 Agent 是独立可插拔的功能单元。

🤖

内置智能体

内置 Chat Agent（对话）、Task Agent（任务调度）、Data Agent（数据分析），支持 DashScope 大模型接入。

🔗

事件总线

系统级事件总线，模块间通过 publish/subscribe 解耦通信，支持 once 监听和自动清理。

🔒

中间件管道

JWT 认证、RBAC 角色鉴权、请求日志等中间件统一管道管理，开箱即用。

💾

持久化存储

轻量级 JSON 持久化存储，提供 CRUD + 查询 + 自增 ID 等操作，可替换为真实数据库。

📊

管理控制台

内置后台管理面板，Agent 管理、用户管理、系统日志、数据统计一应俱全。

系统架构

aiAgentOS/
├── core/                // 系统内核
│   ├── event-bus.js     // 发布/订阅事件系统
│   ├── store.js         // 持久化数据存储
│   ├── agent-registry.js // Agent 生命周期管理
│   └── middleware.js     // 认证 / 角色 / 日志
│
├── agents/              // 内置智能体
│   ├── chat-agent.js    // 对话式 AI
│   ├── task-agent.js    // 任务调度器
│   └── data-agent.js    // 数据分析
│
├── services/            // 业务服务
│   └── auth.js          // 注册与登录
│
├── api/                // REST API 路由
│   └── routes.js
│
├── public/              // 前端页面
│   ├── index.html       // 首页
│   └── pages/           // 管理页面
│
└── app.js               // 入口文件

// 模块接口 — 每个 Agent 都需实现:

  id: 'my-agent',
  name: '我的智能体',

  async execute(input, context) {
    // context: { store, events }
    return { result: '完成' };
  }
